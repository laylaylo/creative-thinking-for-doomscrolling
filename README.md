# Installation Guide

## üîµ For Google Chrome (and Edge/Brave)

### 1. Open Extensions Manager
Open Chrome, type `chrome://extensions` in the address bar, and press **Enter**.

### 2. Enable Developer Mode
Toggle the switch in the top-right corner labeled **Developer mode** to **ON**.

### 3. Load the Extension
Click the **Load unpacked** button (top-left). Select the **entire folder** containing your extension files (`chrome-extension`).

---

## üü† For Mozilla Firefox

### 1. Open Debugging Page
Open Firefox, type `about:debugging` in the address bar, and press **Enter**.

### 2. Select "This Firefox"
Click on the **This Firefox** option in the sidebar on the left.

### 3. Load the Extension
Click the **Load Temporary Add-on...** button.

### 4. Select the Manifest File
Navigate to your extension folder and select the `manifest.json` file specifically.
*(Note: In Firefox, you must reload the extension this way every time you restart the browser).*
